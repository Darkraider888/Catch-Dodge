# Catch & Dodge (Offline)

A minimalist, arcade-style survival game built with vanilla JavaScript and HTML5 Canvas. Catch the coins to score points and dodge the bombs to stay alive!

## ✨ Features
- **Responsive Design**: Playable on both Desktop and Mobile.
- **Offline Capable**: Entirely self-contained in a single file with no external dependencies.
- **Progressive Difficulty**: The game gets faster and bomb frequency increases as your score grows.
- **High Score**: Saves your best performance to local storage.
- **Synthesized Audio**: Uses Web Audio API for sound effects (no .mp3 files needed).

## 🎮 How to Play
- **Catch Coins**: +10 points.
- **Avoid Bombs**: Lose a life. You have 3 lives total.
- **Controls**:
  - **Keyboard**: `Arrow Keys` or `A/D` to move. `P` to pause.
  - **Mouse/Touch**: Click/Tap and drag to move the paddle.

## 🛠️ Development
Since this project is built with pure HTML/JS, you don't need a build process. 
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/catch-and-dodge.git`
2. Open `index.html` in any modern browser.
